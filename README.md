## 📌 About this project

This application provides a simple way to browse content from the CMU Database Group YouTube channel. The repository contains a complete record of 585 videos and 35 playlists. These materials represent 11 years of database systems education. You can search through lectures on topics like SQL, PostgreSQL, and general database architecture without using a web browser.

## 🛠️ System requirements

This software runs on any modern Windows computer. Ensure you have the following items before you start:

*   Windows 10 or Windows 11
*   An active internet connection to stream the videos
*   At least 50 megabytes of free space on your hard drive

## ⚙️ Setting up the application

After you download the file, move it to a folder where you want to keep the application. Follow these instructions to prepare the tool for use:

1.  Right-click the ZIP file you downloaded.
2.  Choose "Extract All" from the menu that appears.
3.  Select a destination folder or keep the default choice.
4.  Click "Extract" to unpack the files.
5.  Open the newly created folder.
6.  Look for a file named "index.exe" or "start.exe" inside the folder.
7.  Double-click this file to launch the application.

If Windows shows a security warning, click "More info" and then "Run anyway." This happens because the application is a private tool and not a commercial product from a large store.

## 🔎 Using the index

The application opens a main window that displays a list of all available videos. You can use the search bar at the top to filter videos by name or topic. 

*   To watch a video, click its title in the list.
*   The application launches the video inside your default web browser.
*   Use the sidebar to filter content by specific playlists. 
*   You can sort the videos by date to see the most recent lectures first.

## 📡 Features

The application provides a clean interface for database students and professionals. Key features include:

*   **Complete searchable index:** Access all 585 videos without scrolling through YouTube.
*   **Offline metadata:** Browse video titles and descriptions even if your network connection drops.
*   **Categorized playlists:** Study specific topics like query optimization, storage engines, or indexing strategies.
*   **Topic navigation:** Filter by specialized tags including PostgreSQL or SQL.

## ❓ Frequently asked questions

**Do I need an internet connection?**
Yes. While the text of the video titles saves on your computer, the actual video files live on YouTube. You need a connection to view the presentations.

**Is this software free?**
Yes. This project is open for anyone to use.

**How do I update the index?**
If you want the newest lectures, return to the link provided in the download section to get the latest version of the folder. Overwrite your current folder with the new files.

**Does the software track my viewing habits?**
No. The application is a static list. It does not send information about your activity to anyone.
