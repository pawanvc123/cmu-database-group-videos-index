## Description

We had the pleasure of welcoming Xiang Fu, co-founder of StarTree, to present Apache Pinot to the CMUDB class. He gave an insightful talk on how Apache Pinot handles real-time OLAP and supports AI-driven features like vector search.

Fun fact: LinkedIn uses Apache Pinot to power its metrics. So every time you view analytics on LinkedIn, you’re seeing Apache Pinot in action.
